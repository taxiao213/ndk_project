知识点：

* 创建 c++ 工程
* jni 写法
* java 调用 c++
* c++ 回调数据给 java
* ffmpeg 编译和导入
* ffmpeg 解码音频文件 
* OpenSLES 使用
* soundTouch 导入和使用(变音 变调)
* 录音音乐
* 剪切音乐
* pcm 数据分包处理
* pcm 数据使用 mediacodec 编码成 aac 文件
* 播放 .ape 文件崩溃问题 一个 AVPacket 会有多个 AVFrame
